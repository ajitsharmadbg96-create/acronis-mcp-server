Acronis API credentials

From the Acronis Management Portal: Settings → API clients. The datacenter URL is whatever shows in your browser's address bar while logged into the portal.

Static Group → Device Mapping

Acronis's public API doesn't expose which devices are in which static group, so this is entered manually as the reliable source of truth — once filled in, it's used everywhere (dashboard, exports) instead of best-effort auto-detection. One group per line, format: GroupName: device1, device2, device3
